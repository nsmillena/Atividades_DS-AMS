HTML

O HTML é uma linguagem de marcação utilizada na construção de páginas na Web.

Descrição da estrutura básica HTML:

Doctype

Código obrigatório, utilizado na primeira linha do documento, que é usado como uma instrução para o navegador e outros programas que possam ler o site que o código apresentado é um HTML. Dessa forma, eles sabem o que fazer para mostrar seu site da melhor forma possível.

HTML lang

Ajuda a definir o idioma de um elemento: a língua em que elementos não-editáveis são escritos, ou a língua em que elementos editáveis devem ser escritos pelo usuário.

Head

Contém informações que não são transpostas visivelmente para o usuário/leitor do documento. São dados implícitos, de uso e controle do documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados. Nela há:

  • Meta charset="UTF-8"

    • A tag "meta" é capaz de definir qualquer tipo de informação(metadados) em HTML. Essas informações podem ser o título da página, sua descrição, seu nome e quaisquer outras informações a respeito de uma página que deva ser informada ao navegador.

    O atributo "charset" define que a informação a ser comunicada pelo atributo meta é um formato de codificação.

    O elemento "UTF-8" é que caracteriza a informação que está sendo comunicada ao navegador. Neste caso, ao utilizar a tag meta charset="UTF-8”, estamos informando ao navegador a informação de que a codificação daquela página é a UTF-8, o padrão de codificação mais utilizado, também chamado de Unicode — ou Conjunto de Caracteres Codificados Universal.

  • Meta name="viewport" content

    • A tag recebe o atributo name com valor viewport que determina que teremos o controle da janela de visualização. O atributo content recebe parâmetros que podem determinar a largura do viewport, a escala inicial para zoom, o valor máximo e mínimo de escala e até impossibilitar o usuário de efetuar zoom.

  • Title

    • É obrigatório e define o título do documento.

    Body

    Trata-se do documento em si, ou seja, a informação legível para o usuário/leitor do documento. É todo e qualquer texto que se deseja apresentar, assim como toda e qualquer forma de mídia de saída. Dentro do elemento "body", a estrutura de página terá os elementos semânticos da construção da página, onde serão declarados e identificados cabeçalhos, rodapé, conteúdo principal, etc. Na prática, o corpo do documento é delimitado pelo par de tags "body".
