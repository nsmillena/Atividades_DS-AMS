HTML

HTML é uma linguagem para a estruturação e apresentação de conteúdo para a World Wide Web

HTML 5 é a mais recente evolução do padrão que define a linguagem HTML

Esta versão traz importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas de novas funcionalidades como semântica e acessibilidade

Possibilita o uso de novos recursos antes possíveis apenas com a aplicação de outras tecnologias
Muitos recursos do HTML 5 tem sido construídos com a intenção de serem executados em dispositivos móveis como smartphones e tablets

História

Entre 1991 e 199, o HTML se desenvolveu a partir da versão 1 até a versão 4. Em 2008, a primeira versão do HTML 5 foi lançada.

As primeiras versões do HTML foram definidas com regras sintáticas flexíveis, o que ajudou aqueles sem familiaridade com a Web.
